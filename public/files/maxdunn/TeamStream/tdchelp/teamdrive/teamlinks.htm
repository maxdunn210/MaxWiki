TeamLink Overview

The easiest way to share files with others is with TeamDrive's TeamLink feature. TeamLinks enables users to link to and work on original documents, rather than emailing documents to one another. TeamLinks eliminates version control issues and can be easily inserted into email, web pages, manual, etc. Then, when a user clicks on the TeamLink, the file is automatically opened and ready for collaborative editing. If TeamDrive is not installed on user's drive, it will automatically self-install at that time.

TeamLink Server

In order to create a TeamLink, you must first specify a server that supports TeamLinks in the Store TeamLinked files on option in the TeamLinks tab.

Creating a TeamLink

To create a TeamLink, right-click on a file in Windows Explorer and select TeamDrive, Create TeamLink. 

Alternately, choose the Create TeamLink button on the TeamLinks tab.

Tip: You can TeamLink several files at a time.

Tip: You can TeamLink files from any drive, including your local drive.

TeamLink Options

TeamLinked files are copied to a special folder on the server specified on the TeamLinks tab. This is where you specify which folder to use and whether it should be for Read-Only Access or Read-Write Access. You can also specify whether an email should be created or if the TeamLink should just be copied to the clipboard.

Receiving a TeamLink

When a TeamLink is received, the recipient will click on the link and a browser window will appear that looks something like this.

If TeamDrive is not installed on the recipients computer, this page will show instructions on how to install it. Otherwise, in a few moments the document will be opened automatically.

Tip: The recipient can click on the same TeamLink again in the future and the most current version of the document will be opened.

Managing TeamLinked Files 

To manage your TeamLinked files, select the Manage TeamLinked Files button. The TeamLinked files can be edited or deleted.

When you created a TeamLink, a copy of the file was placed in this special TeamLink folder so your original copy will not be changed. To update the original document, copy the version in the TeamLink folder back over it.
