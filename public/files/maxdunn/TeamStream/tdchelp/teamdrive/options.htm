Options Tab

There are several options you can use to control the way TeamDrive works. These are location on the Options tab.

Status Monitor Settings

Use these to control when the status monitor appears. The choices are:

  • On errors only
  • When necessary
  • Always

The When necessary option will show the status monitor when the computer is waiting for a file to be downloaded. The Always options will show the status monitor whenever there is any activity on the connection.

Proxy Settings

TeamDrive will normally detect and configure any necessary parameters for proxies or firewalls. However, this can take a little time and there are cases that cannot be handled automatically. This options allows you to set the proxies settings manually.

When first installed, all protocols will be set to Auto-detect on connection. Once a connection has been made, it will be set to either Direct connection, no proxy or the actual proxy address.

To set a proxy address manually, click the checkbox until a solid checkmark appears and then enter the proxy address. If you would like to have TeamDrive run the auto-detect again, click the checkbox until a gray checkmark appears.

ZoneAlarm Configuration

If you are using ZoneAlarm you will need to make few extra configuration steps to let TeamDrive work with this firewall.

Normally ZoneAlarm blocks low-level access to the remote servers. To let TeamDrive access a server you should add it to the list of permitted servers in ZoneAlarm. To do this, follow these steps:

  1. Open main window of ZoneAlarm and select Security tab.
  2. Click the Advanced button. ZoneAlarm Pro users: select LocalZone Contents tab.
  3. Click the Add button and select Host/Site.
  4. Enter a description and the host name of the server you want to access with TeamDrive. For example, if you wanted to access MyDocsOnline you would enter webfolders.mydocsonline.com as a host name.

To enable access to a server on your LAN you may select IP Address option and enter the IP address, for example 192.168.0.1.

Miscellaneous Settings

The Open Explorer window after connection check box is on by default. If you do not want the Explorer window to appear each time, uncheck this box.

Cache Settings

TeamDrive caches files and directory listings on the local hard drive for faster access and for Offline work. For best performance, set the Cache size to at least as large as all the files you normally access with TeamDrive.

Sometimes you might want to clear the cache so that your next directory listing is up-to-date or so TeamDrive will download a recently changed file. You might also want to free up some space on the local drive. For this, use the buttons Clear Cached Files and Clear Cached Directories.
