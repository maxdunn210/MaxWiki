Overview

TeamDrive offers several advanced features for collaboration among team members. They are folder sharing and file locking.

True team collaboration means that you often need to work with your colleagues on the same set of documents, pictures, presentation slides, source code modules, etc. For this purpose you would need to share storage space with the members of your team and put your shared files there.

One option would be to use a single account, and thus one TeamDrive service, for all members of your team. The drawback of this method is that all members will have equal rights for accessing shared documents and then TeamDrive will be unable to track down and control the way they access the documents and protect the documents from accidental changing or deletion.

TeamDrive supports a much better way of sharing data with your team members - you can always share a file or folder with a certain member of your team granting them full or read-only access to it. When you share a folder from your personal account the designated people can access its contents and probably change it, if you allow them to do so, while you retain the ownership and control.

Another way of this powerful mechanism of sharing is to set up a central account and share specific folders with personal account of team members. This way team members will be able to contribute to the contents, while TeamDrive ensures the consistency of the shared data.

Note: In order for sharing and locking to work, it must be supported on the server. The TeamDrive server supports all off these features while other servers may not support all these features.

Sharing Folders

To share a folder, browse to it in Explorer and right-click on it. Then select TeamDrive then Sharing.

Note: You can only share folders, not individual files.

The sharing dialog box will appear so you can add users and select how much access you would like them to have.

In the Share with field, type the name of the user that you want to share with. Then select Read-Only Access or Full Access and press the Add Sharing button.

To stop sharing, select a name and click Stop Sharing. You also can stop all sharing by clicking Unshare All.

If you would like to change the access rights of an existing user, type their name again in the Share with field, select the new access method and select Add Sharing.

When done adding or deleting users, select Ok.

Locking Files

When you lock a file, no one, except you, is able to change or over-write it until it's unlocked. While one team member is working on a shared file it remains locked, so that other team members cannot accidentally corrupt it. They still have read-only access to the file for viewing or printing.

To manually lock one or more files, select the files in the Windows Explorer and right-click on it. Select TeamDrive then Lock Files. 

Note: Only files can be locked, not folders. 

Tip: You can lock or unlock multiple files at the same time.

The locking dialog will appear so you can specify how long you would like to lock the file. This feature makes sure that if someone doesn't unlock a file, other users will be able to access it after the locking period expires.

Unlocking Files

To unlock files, follow the same steps above but choose TeamDrive, Unlock Files from the right-click menu.

Checking Status

TeamDrive's shell integration feature allows you to see sharing and locking status of files right in the Windows Explorer. When applicable TeamDrive shows additional Status column in detailed file view. This column contains status information.



For shared folders the status information shows Master-Shared for the topmost shared folder and Shared for all folders underneath it. It will also say Owner if you are sharing the specified folder with someone.

For locked files it will show the name of the person who locked it.
