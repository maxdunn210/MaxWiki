Requirements

  • Windows 95/ 98/ ME/NT/2000
  • Internet connection
  • 32Mb RAM
  • 10Mb free disk space

Installation

1. Click on the link of the TeamDrive program that you want to install.

2. Select "Run this program from its current location" in the File Download dialog.

3. When the Security Warning dialog appears, select "Yes" to install and run the installation program.

Alternate installation note: You can also choose to download the installation program and then run it  from the local hard disk.

Upgrading a Previous Version

TeamDrive is backwardly compatible and can be installed over previous versions to save the setup information.

Running TeamDrive

TeamDrive will be added to the StartUp group so it will start automatically. To open TeamDrive, click on the icon in the system tray 

Uninstall

To uninstall TeamDrive, go into the Windows Control Panel, choose Add/Remove Programs, scroll down to TeamDrive and select the Add/Remove button. Afterwards, reboot your computer to complete the process.
