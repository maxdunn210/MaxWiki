What "Service" Means

TeamDrive uses the word "Service" to refer to a particular account on a particular server. If you have more than one account on a server, you can setup a different Service for each account. Or if you have accounts on different services, you can setup a different Service for each server.

Pre-Configured Services

If you downloaded TeamDrive from you service provider's web site, it will be pre-configured for that service provider. In this case, you only need to enter your username and password and select the Connect button.

Service List

If you downloaded the general TeamDrive version from the TeamStream site, there will be a list of services that TeamDrive supports. Select the one you would like to use and enter your username and password and select the Connect button. 

Signing Up For A Service

If you don't already have an account with a service, you will need to signup for one. This can be done by selecting the service you want to signup with and selecting the Signup for Service option.

New Service Setup

TeamDrive can also be used with other services, or your own server. It supports FTP, WebDAV and TeamDrive servers. To add a new service, select the Service button and then the New Service item.

Service Properties

The main Service Property windows shows the basic information needed to connect to the service.

Service Name

You can choose a name for your team workplace that you should enter into this field. For instance, the name of one of the pre-defined services is Driveway and it will let you connect to your Driveway account.

Every service should have unique name. If you choose a name that is already taken by another service TeamDrive will adjust it automatically.

Username And Password

TeamDrive needs to know your identification information for this account to be able to gain access to your data. Enter your username and password here to save them along with the rest of service properties.

In case you don't want to store your password permanently you may leave these fields blank or supply only your username.

Protocol

You are asked to choose from several possible options for the desired protocol. These options are:

Auto If TeamDrive recognizes the server type, it will automatically choose the protocol
FTP when connecting to a FTP server
TeamDrive when connecting to a TeamDrive server
WebDAV when connecting to a WebDAV server

Server

Enter a name of the server you are connecting to - for instance:

ftp.example.com for a FTP server
www.example.com for a WebDAV server

Advanced Service Properties

Port

Normally you do not need to fill in the port number so that TeamDrive will use the default port corresponding to the selected protocol. For example port 80 will be used for WebDAV connections and port 21 for FTP connections. However, if the server uses non-standard ports, you can specify it here.

Server Directory

If you usually use just one subdirectory of the server, you can specify the path here and TeamDrive will connect you directly to that directory when you connect.

Do not adjust server time to local time

TeamDrive will normally determine the time difference between your computer and the server and then use this to adjust all file times displayed to the local time. However, if you wish to turn this off, you can select this option and then all file times will be shown with the server time.

Bypass proxy server for this connection

In a normal case you will not need to change the state of this check box. When the box is unchecked TeamDrive will use its default proxy settings for this connection. The only case when you will want to check this box is when you need to bypass the default proxy server specifically for this service. 

Sharing Level

TeamDrive caches directory listings and files for better performance. On the default Light setting, TeamDrive will cache the directory listing for up to 5 minutes. This means that if someone else changes a file, you might not see the changes for awhile.

If you are the only person using an account, you can set Sharing level to None for best performance. If however, many people are sharing an account or are changing files frequently, and you want to be able to see the changes more quickly, you can set this option to a higher level. The Heavy option will slow down performance somewhat but will check very frequently for updated files.

Setting up WebDAV Servers

Some WebDAV servers require a specially-formatted path to access the user's files. Enter the path in the Server Directory field. If the path should contain the username you may use #U notation and it will be replaced by TeamDrive with the actual username.

If your username is myname and the URL to access your files is www.mywebdav.com/~myname, then you need to put ~#U as your server directory path.

Connecting

Once you have signed up for an account, selected or entered the service information and entered your username and password, you are ready to connect. To do this, select the Connect button.

Tip: To save your password so you don't have to re-enter it each time, check the Save password option.

Tip: If you want TeamDrive to automatically connect to this service each time it starts, check the Reconnect automatically option.

Tip: If your server supports SSL secure connections, check the Use secure connection option. This will cause the transfers to be a little slower so if you are not transferring sensitive data and want the maximum speed, leave this option unchecked.
