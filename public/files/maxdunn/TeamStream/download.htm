TeamDrive maps a drive-letter to an Internet server turning the Internet into a virtual hard drive. Then you can use Windows Explorer to easily transfer files or even edit files directly on the server! 

You can use the TeamDrive Client with WebDAV servers, FTP servers, the TeamDrive Server or many Online Storage Services (OSSs).
