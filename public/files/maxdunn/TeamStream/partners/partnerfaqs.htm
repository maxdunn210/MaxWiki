What sets TeamDrive apart from the competition?

TeamDrive’s core functions span the following four different areas: document sharing, document protection, email collaboration and application streaming. Although there are competitors in each of these different areas, no single company offers the entire set of solutions available in TeamDrive.

What is TeamDrive’s key advantage over existing document sharing solutions?

TeamDrive’s key advantage over existing document sharing solutions is that it provides transparent document collaboration features right from the computer desktop. User’s can edit files directly from applications or from Windows Explorer, just like they normally would. Most other solutions like StoragePoint, FreeDrive and Documentum, require users to go to a Web site, download the document, edit it and then upload it again.

What are the TeamDrive’s key advantages over existing e-mail collaboration solutions?

TeamDrive provides a fully integrated system that allows documents to be shared and edited directly through email. Because the method used to do this is very similar to how users currently send document attachments, they will feel right at home with TeamDrive’s email collaboration functions. Other solutions, like those offered by iManage and Click2Send, require the user to use a web browser to either share or retrieve documents.

What are TeamDrive’s key advantages over existing document protection solutions?

Existing document control systems, including Confidential Courier, PageVault and Softlock, provide a good set of document control features. With TeamDrive, documents can be easily distributed and protected with one application.

What are TeamDrive’s advantages over existing application streaming solutions?

Exent, Software Wow and SelectPlay all offer similar functionality to TeamDrive’s application streaming. TeamDrive’s principal advantage over existing solutions is that with TeamDrive application streaming can be tied into document sharing. So if the recipient of a shared document does not have the application to open it, TeamDrive can automatically install the application for them.

What are TeamDrive’s advantages over existing terminal servers?

Existing terminal servers such as Citrix and Microsoft Terminal Server all run the application on a central computer and send just the keystrokes, mouse movements and screen changes back and forth to the desktop computer. This can cause sluggish performance and require multiple expensive servers. Also, if the Internet or server goes down, the applications cannot be used. TeamDrive keeps performance and reliability high by streaming the application to the client PC and running it from there so if the Internet or server is down, the application still be used.

What are TeamDrive’s advantages over existing ASP solutions?

Answer: Existing ASPs must either use terminal servers or have applications that are completely rewritten to run in a web browser. These existing solutions are slow, require retraining and usually do not offer all the features of their desktop equivalents. They also are not as reliable because they will not work if the client, Internet or server is down. TeamDrive does not have these problems because it Internet enables existing applications and runs them on the client PC.

Does TeamDrive compete with OSSs like XDrive or FreeDrive?

No. TeamDrive can be used by these OSSs to offer more functionality to their users. While some OSSs like xDrive or NetDrive already have a tool that provides some of the features of TeamDrive, this just means that the other OSSs will need to acquire similar technology to stay competitive.

Why would a company use TeamDrive instead of a VPN?

VPNs are difficult to setup and maintain, slow, costly and are not good for spontaneous data sharing. The main purpose of a VPN is to connect outside users to the company local area network. However, most outside users just want to quickly and securely access their data from anywhere. This is what TeamDrive allows. TeamDrive can be considered a less expensive and more functional next generation VPN.

Your markets seem fragmented, what is your underlying vision?

Our vision is to provide the Internet infrastructure software needed to connect virtual teams online. Our strategy is the B2B play of selling “shovels to the gold miners” instead of spending a lot of money trying to develop a direct consumer brand.

Isn’t this a crowded space?

Yes, but it is crowded with potential collaborators, not competitors. Even companies that have competing products could use our proprietary TeamDrive technology to enhance their products and services.

Can the TeamDrive technology be easily duplicated?

No. We believe our proprietary TeamDrive technology would be very difficult to duplicate. To the user, it appears simple because it is so well integrated into the operating system. However, there is some very elaborate work going on in the background to make it appear so simple. TeamDrive has been under development by a team of engineers for over 2 years.
