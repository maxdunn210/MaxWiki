TeamDrive turns the Internet into a securely shared virtual hard-drive.

Overview: TeamStream, Inc develops and licenses Internet collaboration infrastructure software to selected strategic partners. TeamStream's solutions enable virtual teams to work together easily and securely, no matter where in the world the team members are located. TeamDrive, the company's proprietary flagship software product, turns the Internet into a virtual hard-drive to provide fast, easy and secure Internet collaboration.

Company: TeamStream is focused on developing and customizing applications for other businesses. TeamStream's main strength is its development team located in the Czech Republic where development costs are low and development quality is high. TeamStream is able to provide customized solutions that significantly enhance our strategic partner's product line.

Products: TeamDrive client maps a drive-letter to an Internet server effectively turning the Internet into a securely shared virtual hard drive. TeamDrive allows any standard Windows program to directly open and edit any document as easily as if it were on the user's private client computer.

Integration: TeamDrive client can work with virtually any Internet server including WebDAV, FTP, HTTP or proprietary servers. This means that the server will not need to be changed to work with TeamDrive. Alternately, the TeamDrive server can be used as the Internet server.

Co-Branding: TeamDrive can be co-branded with the partner's logo and service. When started, the user will see the partner logo at the top and they only need to enter their username and password to connect.

TeamDrive Client Features

  • Fast: Intelligent caching and background transfers make users think they are working with a local drive
  • Easy: Uses a mapped drive-letter so all programs can open documents and edit them in-place on the server
  • Flexible: Works with WebDAV, FTP, HTTP, TeamDrive and proprietary servers
  • Customizable: Can be extended to support other advanced features
  • Co-Branded: Includes partners graphics and preset with partners service information
  • Technical: DLL and VxD installed on client computer, 800k download size
  • Platforms: Win9x, WinME, WinNT, Win2000, WinXP
TeamDrive Server Features
  • Secure: All logins are authenticated and transfers are encrypted
  • Fast: Transfers are compressed for the fastest possible throughput
  • Collaborative: Allows documents to be shared and locked
  • Easy administration: Does not require administrators; if allowed, users can signup themselves and share documents with others
  • Remote Management: Can be remotely managed through a web browser
  • Technical: 1.5Mb download size
  • Platforms: Win9x, WinME, WinNT, Win2000, WinXP, Linux
