TeamDrive Pricing Highlights
Overview
TeamDrive has been priced to be cost effective for both small and large installations.
Increasing Users
Users can be increased at any time. For instance, if the 100-user license is initially purchased and later 200 are needed, simply purchase another 100-user license.
1000+ Options
After a 1000 user license is purchased, there is the option of buying additional licenses on a per-user, per-month basis. For instance, if a 3000 users need to be supported, one 1000 user license could be purchased and then the additional users would be $4,000 per month. Or, if desired, three 1000-user licenses could be purchased.
Server and Client Advantage

When the TeamDrive Server is used with the Client, the following features are available:

  • Encryption
  • Compression
  • Sharing
  • Locking
  • Versioning
Note: Some of these features may be available with other servers.
Client Only
The TeamDrive Client can be used with FTP, WebDAV or supported Online Storage Services like Xythos, MyDocsOnline, bigVAULT, etc.
Co-Brand Customization
With the purchase of at least a 100-user license, the TeamDrive Client will be co-branded with your graphic and service information for no additional charge. This makes it look like "your" program and allows users to connect to your service by just entering their username and password.
