TeamDrive Private Label

TeamDrive can be private labeled and customized for our partners.

Overview: TeamDrive private label includes all the co-brand customizations plus the program name, icons and help pages can be changed so that it truly looks like your own product. The only mention of TeamStream is a copyright notice on the About tab. TeamDrive can also be customized further, to add support for other advanced features like email links, workflow, locking, versioning, etc.

Information Needed For Private Label

  • Name of the program (for instance, "bigVAULT Drive")
  • 468x60 GIF file to use as the graphic on top
  • 16x16 GIF icon
  • Additional 16x16 GIF icons to show "movement" during transfers
  • 32x32 GIF icon
  • Support email address (for instance, "support@company.com")
  • URL for web help pages (we can host on our site at www.teamstream.com/tdchelp/company if desired)
  • The name of your service (for instance, "bigVAULT")
  • The type of server: TeamDrive, WebDAV, FTP
  • The URL of your server (for instance, "webdav.company.com")
  • For WebDAV, the server directory (for instance, "users/joeuser")
