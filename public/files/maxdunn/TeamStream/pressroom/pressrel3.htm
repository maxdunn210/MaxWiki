TeamStream Releases New Online Storage Service Version of TeamDrive

SAN JOSE, California, July 26 /PRNewswire/ -- TeamStream, an emerging Internet collaboration infrastructure software company, today announced the release of a new Online Storage Service (OSS) version of TeamDrive, the Company’s proprietary suite of software solutions that allow virtual teams to work together over the Internet quickly, easily and securely.The new OSS version of TeamDrive allows users direct access to their online storage or File Transfer Protocol (FTP) accounts. TeamStream plans to offer the new customized OSS version of TeamDrive to OSS providers. OSS providers will be able to use the new version of TeamDrive immediately, without changes to their storage servers.

"We’re pleased to add OSS functionality to TeamDrive,” said Rick C. Garian, TeamStream’s President and CEO. “We expect to release additional key new features, including document sharing, e-mail collaboration, application providing, and document protection, in the near term."

With the new OSS version of TeamDrive, users can map a drive letter to their accounts and run Windows Explorer or Windows Commander to edit or drag and drop files back and forth between computers and online storage (WebDAV) or FTP accounts. In addition to the new OSS version, TeamStream’s TeamDrive technology is applicable for Virtual Private Networks (VPNs), Application Service Providers (ASPs), Enterprise Information Portals (EIPs) and Internet Service Providers (ISPs).
