ENTERPRISE FILE SHARING GETS EASIER WITH XYTHOS’ TEAMSTREAM ACQUISITION
Client Software Provides Cost Effective Solution for Windows Users Today

SAN FRANCISCO, Calif., August 8, 2002 - Xythos Software (www.xythos.com) strengthened its position as the leading Internet File Management (IFM) software provider with its acquisition of San Jose-based TeamStream Software and its top-rated TeamDrive software. The combination of the Xythos WebFile Server and TeamStream’s client software provides a compelling solution for enterprise customers seeking secure file access and sharing features directly from within their current Windows applications. With TeamDrive software, users benefit from access to advanced WebDAV file management features such as versioning and access control lists previously only available via web browser. In addition, users gain automatic offline file synchronization. Enterprise IT departments will appreciate that the TeamDrive client works seamlessly together with the Xythos WebFile Server (WFS) and supports all leading server and storage environments helping improve their return on investment (ROI) from existing network infrastructure.

"Customers want flexibility and safety when sharing files," commented Edward Miller, CEO at Xythos. "With this acquisition, they get to choose the best way to do that depending upon their needs - from within Windows programs on their desktop, or by using a browser. Either way, Xythos’ support for the latest user authentication and Internet security standards will ensure their files are always safe."

Available today, Xythos WFS and TeamDrive-based WebFile Client (WFC) software support virtually any file storage and server environment, providing an option to enterprises unable or not ready to convert all file services to Windows servers. Xythos WFC supplements secure browser-based file access allowing any Windows desktop application to become WebDAV enabled by presenting remote resources as a local drives and adding advanced collaboration features such as, remote file locking and directory, folder and file level read, write and share control to Windows Explorer menus. The WFC also helps traveling users automatically synchronize laptop files with the WFS so that file collaboration can continue even when they are disconnected.

"Enterprise customers shouldn’t be forced to upgrade Windows desktops or be limited to Microsoft-only servers just to enjoy safe file sharing from within their favorite Windows applications," explained Max Dunn, former president and CEO of TeamStream and now Vice President of Client Technology for Xythos. "We make universal and standard access to WebDAV functions in any Windows application possible today and that means users can save time and employers can avoid unwanted software upgrade costs."

What is WebDAV?
WebDAV stands for "Web-based Distributed Authoring and Versioning" and it is a set of extensions to the HTTP protocol which allows users to collaboratively edit and manage files on remote web servers. For more information: http://www.webdav.org/.
