The first in a suite of solutions that allow virtual teams to collaborate quickly, easily and securely over the Internet, TeamDrive offers proprietary technologies that turn the Internet into a virtual hard-drive. The product is available in both client only and client/server configurations. 

The client software can be installed on all Windows platforms and works with WebDAV, FTP, HTTP and proprietary servers. The Server product works with all Windows platforms as well as with Linux. 

Since its inception, TeamStream's products have turned Internet servers into secure, shared virtual hard drives. The products, which feature drive-letter mapping, local caching, 'instant saves', and compression, enable any standard Windows file to be easily opened and edited. They also offer checkout/in and versioning features that facilitate easy collaboration. 

New in Version 3.0

In Version, 3.0, TeamDrive has been expanded to address the needs of mobile users, providing easy and secure access to their files, either at work or while traveling. The offline work capability allows users to continue working on their files when traveling and/or when no Internet connection is available. The next time users connect to the Internet, all changes are automatically synchronized to the central server. SSL secure connections ensure that, regardless of a user's log in location, his or her data will be completely secure as it travels across the Internet. 

With the document linking feature, TeamLinks, mobile users can view and edit original documents, not copies of documents, directly, making and saving all changes on the original. They can communicate with anyone including users that do not currently have TeamDrive installed. 

In lieu of emailing documents, individuals or workgroup members can now click on a link and automatically retrieve and work on the most current versions of documents. Individual users can also access documents from anywhere in the world, as well as working in virtual teams to share documents with other users regardless of where they are geographically. TeamLinks provides a replacement for email attachments and eliminates versioning problems as each recipient can edit the original document

Mapped Drive Letter

To Windows, TeamDrive looks like any other networked drive by operating as an extension of the Windows file system. This allows team members to map drive letters to a TeamDrive server and use them to directly load and save files from any application. Editing is done directly on the TeamDrive server so no copying to the local computer is needed.
