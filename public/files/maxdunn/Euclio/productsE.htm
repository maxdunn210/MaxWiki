TeamDrive
TeamDrive is the software for people who need to share the same documents and are not connected to the same local network LAN.
TeamDrive enables access to the documents saved on the Internet servers (FTP, WebDAW and TeamDrive). It is possible to open, edit, move and copy shared documents (e.g. by Windows Explorer or Windows Commander). Sharing through Internet means that the applications can process the data directly on the remote server and it is not necessary to save the files on the local disk. The work is as easy, quick and comfortable as on the network server in the local network.
Thanks to its built-in security functions the TeamDrive enables the coded data transfers through Internet.

More details about the TeamDrive: http://www.teamstream.com
