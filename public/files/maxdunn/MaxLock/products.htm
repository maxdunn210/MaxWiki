EuCore PCI-M32
32 bit / 33 MHz PCI Bus Master/Target

Fully functional PCI bus Master/Target interface gives possibility to initiate PCI transactions as well as to respond to PCI accesses. User-friendly back-end interface allows simple connection to a user design. Core is delivered in synthesizable VHDL code.

Download!

EuCore PCI-T32
32 bit / 33 MHz PCI Bus Target

PCI Target interface with simple user-friendly interface. Core supports all the Base Address Registers and has capability to initiate Abort, Retry and Disconnect abnormal terminations. Core is delivered in synthesizable VHDL code.

Download!

PCI Prototyping board PC 300

PC300 is a low cost prototyping card, that allows easy evaluation of design examples and fast prototyping of new custom designs with MaxLock's PCI cores.
