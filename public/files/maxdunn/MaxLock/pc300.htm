PC 300

Key features

  •  Standard 32-bit 33Mhz PCI Card
  •  3.3 and 5V PCI 2.2 compliant
  •  Xilinx VIRTEX XCV100-5PQ240 FPGA (100k System Gates) - PQFP240 package allows VIRTEX scalability in chip capacity ranging from 50k system gates for simple applications up to 800k system gates for complex systems.
  •  Xilinx XC9536XL for FPGA configuration from parallel FlashROM
  •  Xilinx XC1801 Serial configuration FlashROM: optional
  •  Parallel Flash up to 256kB - can be used both for application needs and FPGA configuration instead of XC1801
  •  Synchronous SRAM 64kx32: optional
  •  Expansion connector for daughter cards - offers expandability for prototyping additional features
