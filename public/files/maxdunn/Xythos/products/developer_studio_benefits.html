With the Xythos Developer Studio, developers can:

  • Easily Build New Applications on the Xythos Platform: Developers have complete access to modify all features within the Xythos system, including: the web user interface, workflow processes, security models, and all document management functions, enabling them to build new applications easily, leveraging the collaborative features of the Xythos WebFile Server.
  • Reduce Software Development Time: Developer Studio provides extensive sample code, a complete Developer's Guide, and a documented API to accelerate development projects designed for Xythos systems (reduce IT costs). Common Java development tools can also be used helping reduce training requirements.
  • Use the IDE They Are Most Comfortable With: Because of the WebFile Server's standards-based architecture, development for the WebFile Server APIs and the Workflow Designer can be done in Linux, Unix and Windows environments, allowing developers to work within any integrated development environment.
