Customize Content Management Software
Content Management Software APIs

WebFile Server APIs: Xythos WebFile Server content management software APIs allow custom applications to extend the functionality of the Xythos WebFile Server. The APIs offers an unparalleled level of flexibility with its simple, yet powerful set of interfaces and objects. Among other things, the API allows programmers to:

  • Monitor events occurring anywhere on the system
  • Replace the standard encryption with a custom algorithm
  • Control permissions
  • Implement custom security/permissions models
  • Perform actions on resources, with permission checks
  • Create, name, set and delete properties
  • Lock directory quotas so they can't be changed by users
  • Implement custom logic before or after WebDAV requests
  • Incorporate a full-text search engine to search content
  • Create custom logging classes

As a result, developers and partners can customize the Xythos WebFile Server to bring power and intelligence to their own services and offerings.

The Xythos WebFile Server APIs allow developers to integrate a standard set of library services.
