Xythos WebFile Development Suite is the complete set of products for developers to cost-effectively integrate document and file management functionality into enterprise applications.

Learn more about each of the WebFile Development Suite products:

  • Xythos WebFile Server is the content management platform that manages and stores all of the content, documents and files accessed via the Enterprise Document Manager and Digital Locker applications. Webfile Server is developed in Java and is a robust implementation of the (WebDAV) standard - Web Distributed Authoring and Versioning protocol .
  • Xythos Drive turns any Windows desktop application into a WebDAV-powered document management and collaboration tool and provides the offline collaboration capabilities that mobile professionals demand.
  • Xythos Developer Studio includes all of the development tools that an application developer needs to create or customize a Xythos document management installation.
