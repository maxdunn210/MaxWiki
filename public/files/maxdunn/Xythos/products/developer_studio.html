Developer Studio

Content Management Software Installation
Customized Document Management Software

Xythos Developer Studio includes everything a developer needs to create or customize a Xythos content management software installation. It provides a developer, test and backup instance license for the Xythos WebFile Server and the Xythos Drive.

The Developer Studio's customized document managment software contains APIs that allow custom applications to extend the functionality of the Xythos WebFile Server. An application built on WebFile Server may be as small as a simple program run nightly to clean up trashcan files older than a certain date or it might be as complicated as a full graphical user interface, like the WebUI code, exposing much of the functionality of WebFile Server to clients.
