Xythos Drive Features

WebDAV Server Technology
Secure File Access
Document Version Control
Information Sharing
Offline Synchronization


  • Test Drive
    Xythos Enterprise Document Management Suite free for 30 days.

    Map Letter Drives: Use Xythos Drive to connect to or map a network drive to a folder on WebDAV server technology, effectively turning the Internet into a secure, shared virtual hard drive. The Xythos network drive behaves similarly to a Windows network-mapped drive, except that it uses HTTP and WebDAV protocols. This permits any Windows program to open and edit documents and files stored on a server as easily as if they were on the user's own computer. Right-click the file to access standard Explorer commands such as Edit, Print, Copy, Delete, Rename, etc. Right-click the right Explorer panel to create new files or folders on the server.
  • Improved Collaboration from the Desktop: Xythos Drive includes high-level collaboration features that Windows alone can't offer. Right-clicking on a file in Windows Explorer enables users to easily control access rights, file locking and document versions; users can also view a document's version history, check a version in or out, or mark it for offline synchronization without ever having to leave the desktop environment and applications they use every day.
  • Offline Synchronization: Xythos Drive provides offline file synchronization for traveling users. Users can select any number of directories or files they want to access when they are disconnected from the Internet; these files are saved locally, permitting users to edit and manage these files while they are offline, and then automatically uploaded next time the user connects.
  • Enterprise-Level Security: Xythos Drive supports SSL 128-bit and higher encryption or VPN access, ensuring that all file transmissions between client and server remain protected. Xythos Drive is also designed to authenticate against the server's user and group model or LDAP, ensuring secure file access only to those who are permitted. Authorization is maintained by the server's access control, allowing users to see only those directories to which they are permitted.
  • Intellittach Technology: Xythos Drive uses Intellittach technology to automatically replace email file attachments with secure links, improving the security of files being shared. Users don't have to copy files to a server so that they can be safely accessed by others; it is automatic. Only one copy of the document is saved on the server, regardless of the number of recipients. Xythos Drive also permits server and directory locations to be pre-selected ensuring that files are automatically saved in the correct server location, helping meet compliance regulations. Stored files can inherit access permissions from parent directories or folders, increasing security. There is no need to give special instructions when sharing information on Intellitach technology. The attachment looks like a regular attachment. Your recipient can view or download the attachment by double-clicking it.
  • Backup: Xythos Drive includes an automatic online backup to provide secure protection of user directories which are not already stored on a server such as, My Documents. IT departments can pre-set a user's backup destination or users can select their own location. Finally, files can be backed up on either an on-demand or scheduled basis.

Xythos Drive gives users the ability to access Enterprise Document Manager features from any Windows desktop application.
