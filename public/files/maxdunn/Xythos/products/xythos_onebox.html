Xythos Google OneBox Module

Google Search Integration
Document & File Security

Google and Xythos have been integrated to provide search results from the Xythos repository and Google Search Appliance (GSA) results in a single view. This Google search integration, via a Google OneBox module, grants users real-time secure access to their information displayed on the GSA results page. The view allows users to see search results from multiple enterprise systems on a single page. Since Xythos handles the document and file security, a search will only return results for documents that the user has access to.

Xythos OneBox gives users real-time secure access to their information displayed on a Google Search Appliance page
(Click sreen shot for larger image)

Like all Enterprise OneBox modules, the results are displayed at the top of a Google GSA’s results page. Following Google OneBox for Enterprise design principles, the results are brief with only the most relevant results displayed. A link is provided back to Xythos’ web UI for complete results.

The Xythos OneBox module works with all of Xythos’ server products: Enterprise Document Manager, Digital Locker, and WebFile Server and is provided for free on Xythos’ developer website. The Xythos OneBox module also provides the same flexibility as other Xythos products. The source code to the module is provided so you can customize it as needed, using the flexible Xythos platform APIs.

More information and the source code for Xythos OneBox can be found in the project forums at http://developer.xythos.com.
