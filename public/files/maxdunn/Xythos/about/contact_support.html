Please note that you will only be able to access the Xythos support site if you are a current Xythos product customer (not a Xythos on Demand customer) and have been given a valid login user name and password.

If you require Xythos product support, please help us facilitate the problem-solving process by providing the following information regarding your issue. Please use the below problem report template to facilitate email and phone support requests:

  • The major and minor version numbers of the product that is in production at your installation. e.g. 4.1.23.6
  • The name and version of the web server(s) and/or application servers used in combination with your Xythos system.
  • The data repository with version number that powers the back end of your WFS installation (RDBMS, LDAP, etc).
  • A detailed description of the problem.
  • Any exception or log messages produced by the Xythos system, as a result of the problem encountered. If you have configured the app server to write its output to a log file, please provide Xythos support with this log file.
  • The ticket number of the issue, if you have been asked for more information or are replying to an email from Xythos Support.

An email to support@xythos.com containing these facts will greatly help to expedite a reply to your questions.
